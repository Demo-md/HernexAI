# Insightsnode Footer Credit

The footer renders the semantic text `Developed & Powered by Insightsnode` without a link because no verified official Insightsnode URL is stored in the repository.

`setupInsightsCredit` uses an IntersectionObserver and one short GSAP sequence: mask reveal for the prefix, letter entrance for `Insightsnode`, then rest. Reduced-motion mode leaves the complete credit static. No canvas, 3D text, perspective, or continuous rotation is used.
