# HernexAI Browser QA Matrix

Required viewport widths: `1440`, `1280`, `1024`, `768`, `430`, `390`.

At each width check initial load, fonts, headline/CTA alignment, navigation, major routes, forms, hover/focus, scroll and reverse scroll, sticky/pinned content, animation final states, reduced motion, console/page errors, horizontal overflow, and mobile-safe overlays. Capture baseline and final screenshots for visual work. Record unavailable routes or absent frontend as failures, not passes.
