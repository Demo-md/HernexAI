# Skill Activation Order

Use this default sequence when responsibilities apply:

1. `design-reference-analyzer`
2. `hernex-brand-guardian`
3. `premium-frontend`
4. `anti-generic-ui`
5. `astro-architecture`
6. `tailwind-v4-design-system`
7. `gsap-timeline`
8. `gsap-performance`
9. `motion-accessibility`
10. `astro-view-transitions`
11. `spline-threejs-integration` only when 3D is explicitly requested or justified
12. `frontend-design-critic`
13. `agent-browser-verify`
14. `visual-regression-qa`
15. `playwright-interactive`

Combine skills only when their responsibilities are relevant. The analyzer gates implementation when a reference is supplied. Critique is review-only unless the user explicitly asks for changes. Browser and visual QA are required before completion for frontend work.
