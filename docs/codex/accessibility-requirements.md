# HernexAI Accessibility Requirements

- Use one logical `h1`, ordered headings, landmarks, semantic links, and real buttons.
- Keep essential content visible without JavaScript or animation.
- Make every menu, disclosure, slider, dialog, and chatbot keyboard operable.
- Provide visible focus, accurate names, `aria-expanded`/`aria-controls` where needed, and focus restoration.
- Support Escape for dismissible overlays and do not trap focus outside an active dialog.
- Respect `prefers-reduced-motion`; remove non-essential loops, scrubbing, and smooth-scroll coupling.
- Do not use custom cursors or hover-only meaning on touch devices.
- Provide useful alt text for informative images and empty alt for decorative images.
- Check contrast, touch target size, zoom/reflow, no horizontal overflow, and screen-reader order.
