# HN Bot Asset Requirements

No authorised mascot asset exists in the repository as of Phase 2. The current original CSS 2.5D bot is the active asset and fallback.

Before a future real-time model is introduced, require:

1. An original or commercially authorised HN Bot visual design approved for HernexAI.
2. Source/ownership documentation and licence terms for all geometry, textures, rig, animation, fonts, and audio.
3. A GLB preferred over separate GLTF assets, with a documented polygon count and texture inventory.
4. Texture dimensions suitable for a small floating launcher; avoid large 4K maps and realtime reflections.
5. Idle, attentive, processing, success, error, and resting poses that remain professional and subtle.
6. A static CSS/image fallback retained beside the model path.
7. Evidence that a lazy-loaded, DPR-capped renderer fits the frontend performance budget on mobile and desktop.

Do not use public robot models, another company's Spline scene, generated mascot imagery without rights confirmation, or copied robot geometry.
