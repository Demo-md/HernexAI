# HernexAI Brand Rules

- Use the verified navy-and-gold direction from current project briefs unless an approved brief changes it.
- Keep the tone premium, modern, professional, human-led, and clear.
- Position AI as an enabling capability inside a broader marketing/growth system.
- Use actual logos, assets, and type choices from the repository when they exist.
- Avoid generic neon cyberpunk, rainbow gradients, competitor mimicry, and claims presented as proof without evidence.
- Treat missing assets, missing copy, and missing proof as explicit gaps.
