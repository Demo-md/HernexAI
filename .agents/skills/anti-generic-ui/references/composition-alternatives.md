# Composition Alternatives

Use these as prompts, not templates:

- Replace a card grid with a typographic list, rule-separated rows, or a single editorial feature.
- Replace floating cards with a structured overlap tied to a focal media frame.
- Replace a feature matrix with a sticky narrative and progressive proof.
- Replace repeated service cards with a timeline, index, or horizontal chapter rail.
- Replace decorative blobs with a diagram that explains flow, relationship, or state.
- Replace centered sections with a two-column statement/detail layout when the content has hierarchy.
- Replace a generic stats row with one meaningful proof statement and supporting context.
